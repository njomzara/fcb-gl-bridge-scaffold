# /FCBP/GLT_DCL - CDS Access Control Package

This package owns DCL scaffolds for CDS view access control across the GL Bridge support and administration surfaces.

## DCL Artifacts

- `/FCBP/DCL_GLT_TRANSFER`
  - Access control for transfer and monitor-facing transfer projections.

- `/FCBP/DCL_GLT_PACKAGE`
  - Access control for package evidence projections.

- `/FCBP/DCL_GLT_VALIDATION`
  - Access control for validation runs and findings.

- `/FCBP/DCL_GLT_MAPPING`
  - Access control for mapping evidence.

- `/FCBP/DCL_GLT_ADAPTER`
  - Access control for adapter attempts, target references, and adapter health projections.

- `/FCBP/DCL_GLT_CONFIG`
  - Access control for configuration and policy administration projections.

- `/FCBP/DCL_GLT_AUDIT`
  - Access control for audit and support audit projections.

- `/FCBP/DCL_GLT_SOURCE_READING`
  - Access control for source reading and source trace projections.
